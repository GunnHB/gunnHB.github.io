## 템플릿
`템플릿template`은 <u>원하는 부분에서 타입을 원하는 타입으로 지정할 수 있는 기능</u>입니다.

```c++
#include <iostream>

// template <typename 원하는 이름, typename 원하는 이름, ...>
// typename 대신 class를 사용해도 된다.
// typename <typename 원하는 이름, class 원하는 이름, ...> 가능

template <typename T>
T Plus(T num1, T num2)
{
	return num1 + num2;
}

int main()
{
	std::cout << Plus<int>(10, 20) << std::endl;				// T가 int 타입화
	std::cout << Plus<float>(3.14f, 5.11f) << std::endl;		// T가 float 타입화
	std::cout << Plus(4.66f, 7.23f) << std::endl;				// 타입을 생략하면 인자의 타입으로 결정됨

	// 만약 템플릿을 사용하지 않았다면 위 함수는 타입별로 오버로딩하여 구현했을 것
}
```

기본 타입을 따로 지정할 수도 있습니다.

```c++
#include <iostream>

template <typename T = float>
T Plus(T num1, T num2)
{
	return num1 + num2;
}

int main()
{
	std::cout << Plus(3.14f, 2.55f) << std::endl;
	std::cout << Plus(10, 20) << std::endl;
	std::cout << Plus(10, 20) << std::endl;
}
```
